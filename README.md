🌦️ Weather App

-> Description
A simple weather app that fetches real-time data using OpenWeather API.

-> Features
- Search weather by city
- Displays temperature, humidity, condition
- Clean UI

-> Tech Used
- HTML
- CSS
- JavaScript
- OpenWeather API

>- Setup
1. Get API key from OpenWeather
2. Add it in script.js
3. Open index.html

Get Free API Key
1. Go to: https://openweathermap.org/api
2. Sign up
3. Copy your API key
